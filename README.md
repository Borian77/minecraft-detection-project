
# Minecraft Object Detection Project 

Система автоматической детекции игровых сущностей (мобов) в Minecraft с использованием глубокого обучения.

## Описание
В данном проекте проведено сравнительное исследование двух архитектур детекции объектов на специфичном датасете Minecraft (18 классов мобов: зомби, криперы, животные и др.):
1. **YOLOv8s** (Ultralytics) — современная SOTA-модель.
2. **FCOS** (MMDetection) — anchor-free подход.

## Результаты тестирования

| Модель      | mAP@50 | FPS (CPU) |
|-------------|--------|-----------|
| **YOLOv8s** | 0.787  | 30.11     |
| **FCOS**    | 0.664  | 2.79      |

### Выводы
- **YOLOv8s** признана оптимальной для использования. Она демонстрирует баланс точности и скорости, достаточный для анализа игровой ситуации.
- **FCOS** в текущей конфигурации показала неплохие результаты, но уступающие YOLO. 

## Структура проекта
- `datasets/minecraft/` — изображения и аннотации. Структурированы в подпапки `images` и `labels` для совместимости.
- `artifacts/` — итоговые отчеты (PDF), логи и графики обучения.
- `runs/` — веса обученных моделей (best.pt).
- `fcos_minecraft_3.py` — конфигурация для MMDetection.

## Инструкция по запуску
1. Установите зависимости: `pip install ultralytics mmcv==2.1.0 mmdet`.
2. Подготовьте данные: запустите скрипт конвертации COCO -> YOLO.
3. Запуск инференса YOLO:
   ```python
   from ultralytics import YOLO
   model = YOLO('runs/detect/artifacts/yolo/weights/best.pt')
   model.predict(source='path_to_image.jpg', save=True)
